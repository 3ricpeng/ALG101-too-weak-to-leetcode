# 空間估算

* int: 4 bytes
* double: 8 bytes
* JS 中的 Number：8 bytes

---

# 一百萬個數字

(8 * 1e6) / 1024 = 7812 KB  
= 7.6 MB

---

# 十億個數字

(8 * 1e8) / 1024 = 781200 KB  
= 7600 MB = 7.4 GB

---

# 範圍決定方法

排序十萬個數字，可以全都載入到記憶體  
但如果要排十億個數字呢？
