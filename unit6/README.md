#  Unit6：內建函式做做看

之前不讓大家用這些內建函式，是害怕大家會陷入一個困境  
那就是解題的時候不去想解法，而是想說：「這題可不可以用 xxx 解？」

但本質上他們是一樣的，你用內建函式可以做到的功能，for 迴圈搭配 if else 幾乎都做得到  
因此，這一個單元要來實作這些內建函式，當你能夠實作出他們以後，之後就放心地用吧！
