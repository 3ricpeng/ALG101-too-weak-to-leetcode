
## 字串反轉

let s = ''
for (i from n-1 to 0) do

## 陣列總和

```
let sum = 0
for (i from 0 to n-1) do
  sum += arr[i]
end for
print sum
```


## 找最大值

```
let max = arr[0]
for (i from 0 to n-1) do
  if (arr[i] > max) do
    max = arr[i]
  end if
end for
print max
```
